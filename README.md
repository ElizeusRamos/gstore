# GStore - Sitema Generic Store

Sistema genérico de gerenciamento de loja, dentro principais funcionalidades básicas inclusas:
- Controle de cliente
- Controle de estoque
- Controle de Pedido 

---
Tecnologias utilizadas no projeto
- Java 11
- Apache Maven 3.6
- Apache Tomcat 8
- MySql 8
- Bootstrap v 5.2

---
Ferramentas utilizadas
- Eclipse
- DBeaver
- Notepad++
- GitBash
